# AutoQA-Agent

A LangGraph-powered multi-agent AI system for automated Playwright failure investigation.

Features:

- Log Parsing Agent
- Root Cause Analysis Agent
- Failure Classification Agent
- Evaluation Agent
- Human-in-the-Loop Routing
- Automated Bug Report Generation
- FastAPI Backend
- Dockerized Deployment
